# NBU Course Table to *.ics file Tool

Simply fetch NBU courses table from ehall and generate ics file. 



## Run

```shell
pip install -r dependencies.txt
python LoginForm.py
```



## Build

**Module py2app is NEEDED.** 

```shell
python setup.py py2app
```

Program will be placed at  `./dist` .

# 宁波大学课程表获取并生成日历文件工具

### 功能

轻松获取宁波大学网上办事大厅课程表数据并生成ics文件，可导入手机日历



## 运行

```shell
pip install -r dependencies.txt
python LoginForm.py
```



## 构建

**需要预先安装py2app模块。**

```shell
python setup.py py2app
```

构建的程序在 `./dist` 目录下
